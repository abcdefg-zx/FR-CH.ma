💡 使用提示

  • 免费自动：无需密钥，自动尝试Google/MyMemory等多个引擎
  • Google翻译：免费，质量高，推荐使用
  • DeepL：翻译质量最高，免费版有限额，付费需填Key
  • MyMemory：免费社区翻译，适合简单文本
  • LibreTranslate：开源免费翻译服务
  • DeepSeek/OpenAI：需API密钥，失败时自动降级到免费服务
  • API密钥仅保存在本地浏览器，不会上传到服务器

译文（中文）
翻译结果将显示在这里...

💡 使用提示

  • 翻译方向：可选择"法语→中文"或"中文→法语"
  • 点击"开启字幕"按钮后，浏览器会请求麦克风权限
  • 说话时请保持连贯，系统会自动累积识别结果
  • 点击"清空字幕"可清除之前的识别内容
  • 实时字幕功能依赖浏览器的Web Speech API
  • Chrome、Edge浏览器支持较好
